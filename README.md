#### 本地运行

1. 克隆代码
   ```bash
   git clone https://github.com/octopusYan/alist-gui
   ```
2. 运行
   ```bash
   mvn clean javafx:run
   ```

#### 构建

1. 克隆代码
   ```bash
   git clone https://github.com/octopusYan/alist-gui
   ```
2. 运行
   ```bash
   mvn package
   ```
